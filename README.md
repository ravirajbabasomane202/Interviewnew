# Interactive Interview Preparation Workflow Canvas

Personal web app to build and track interview preparation roadmaps.

## Stack
- React + Vite
- React Flow
- TailwindCSS + Framer Motion
- Monaco Editor
- Node.js + Express
- MongoDB + Mongoose

## Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start MongoDB locally.
3. Start app + API:
   ```bash
   npm run dev:full
   ```

Frontend: http://localhost:5173  
Backend: http://localhost:5000
